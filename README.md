# Suggestions
Players can suggest for your server
# How to use
/suggestions and it open form ui with for suggestions
# Permissions
```
permissions:
  suggestions.command:
    default: true
  suggestions.admin:
    default: op
```
# config
```
---
# suggestions list limit
AllMax: 30

# player suggestions limit
AllMax: 3
...
```
